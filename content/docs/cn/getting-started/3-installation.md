要使用NativeScript-Vue，您需要设置系统以编译应用程序。

## 先决条件

- [Node.js](#nodejs)
- [NativeScript CLI](#nativescript-cli)
- [Windows](#windows) （用于在Windows上开发Android应用程序）
- [macOS](#macos) （用于在macOS上开发iOS和Android应用程序）
- [Linux](#linux) （用于在Linux上开发Android应用程序）

### Node.js

下载并安装最新的LTS版Node.js [https://nodejs.org/](https://nodejs.org/)。重新启动终端并通过运行验证安装是否成功 `node --version`。

### NativeScript CLI

要安装NativeScript CLI，请打开终端并运行：

```shell
$ npm install -g nativescript
```

通过在终端上运行 `tns` 命令来验证安装是否成功。您应该会看到可用的命令列表。

### Windows

检查系统要求并按照Windows上Android SDK设置说明操作：

[高级设置：Windows](https://docs.nativescript.org/start/ns-setup-win)

### macOS

检查系统要求并按照macOS上的Xcode和Android SDK设置说明进行操作：

[高级设置：macOS](https://docs.nativescript.org/start/ns-setup-os-x)

### Linux

检查系统要求并按照Linux上Android SDK设置说明操作：

[高级设置：Linux](https://docs.nativescript.org/start/ns-setup-linux)
