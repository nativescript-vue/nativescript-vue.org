The Button component is used to display a button that can react to a tap event.

---

```html
<Button text="Button"/>
```
[> screenshots for=Button <]

## Props

| name | type | description |
|------|------|-------------|
| `text` | `String` | The text of the button.

## Events

| name | description |
|------|-------------|
| `tap`| Emitted when the button is pressed
